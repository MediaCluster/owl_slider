Users manual
------------

- Create a SysFolder named e.g. “Slides”

- Switch to “List” and create an “owlSlider” object. This object
  contains all necessary settings for the frontend rendering

- Set the title, width, height and all required settings

- Add an “owlSlider Slide”. This object contains all elements of a
  slide.

- Set a title for the slide

- Switch to “Page” and insert the Plugin “owlSlider” to the page and
  select your slider object.
